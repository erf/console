# termio

A minimal lib for console apps in Dart.

No dependencies. Just pure Dart.

## Features

- raw mode
- get cols and rows
- window resize events
- input stream
- buffered input
- basic VT 100 escape codes (move cursor, colors, clear screen, etc)

## Examples (see `example/` folder)

- sweep (like minesweeper)
- snake
- game of life
- simple
- colors
